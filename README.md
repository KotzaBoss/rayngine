# rayngine

## Getting started

```shell
git clone https://github.com/KotzaBoss/rayngine.git
cd rayngine

cmake --workflow --preset=homeworld-build && ./.build/bin/rayngine
```

For details on the presets/workflow see [[/CMakePresets.json]]
